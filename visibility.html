{visibility of system status}

“The system should always keep users informed about what is going on, through appropriate feedback within reasonable time.”

Chat Box Feedback:
On the library page there is an application “meebo”, which allows students to communicate with librarians via real-time web chat. The meebo application has a chat box in which users can input questions and/or comments directed at University Library staff. Before the user types anything, the chat window has instructions, which direct the user how to use the application. The directions read, “Type here and press enter to send a private message.” When the user types in that box, the text automatically disappears.


No Breadcrumb:
This site lacks the use of a breadcrumb. A breadcrumb informs the user where he/she is on the site. Thus by not having a breadcrumb the user has to make an added effort to keep track of their location.


No Highlighting:
The current page the user is on is not highlighted in the navigation, making it difficult for users to know where they are in the website. Typically if there is a navigation bar in which the headings are highlighted as they are moused-over, the heading that is clicked will remain highlighted. This informs the user where they are on the site.
