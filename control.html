{user control and freedom}

“Users often choose system functions by mistake and will need a clearly marked ‘emergency exit’ to leave the unwanted state without having to go through an extended dialogue. Support undo and redo.”

Inconsistent Navigation Bar:
The main page of the University of Louisiana website contains a navigation bar and links to the home page, including a consistent logo which links back to the home page. However, when visiting the “Library” page, there is no longer any resemblance of the navigation bar, and no links back to the home page. There is a very small link back to the home page at the bottom of the page, but it is fairly hidden. The user would expect there to be a way to get back to the homepage via the navigation bar or a picture link, instead the user has to user the browser to return back or spend time looking for the hidden link at the bottom of the page.


Confusing slideshow:
The animated slideshow in on the home page does more harm than good, as it causes frustration for the user as the picture only links to one description. Even though there are multiple pictures that cycle through the slideshow, only the current photo is linked. Therefore, when the user follows the link they are only provided with a single description with no access or ability to see the other pictures and descriptions as seen on the home page. Furthermore, the slideshow on the homepage leads users to an unwanted and unrealistic page. It takes added effort to see other descriptions of pictures. To see other pictures the user has to refresh the homepage until their desired picture is seen and then they have to click on the link. This limits user control and freedom greatly.
